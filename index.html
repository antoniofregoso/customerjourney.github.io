CustomerJourney.js is a set of JavaScript libraries designed to create landing pages, funnels, and touchpoints with one goal in mind: hyper-personalization of the user experience with AI.

The concept is based on macro components:

  1. Approach as a Web App as a system

  2. Each macro component represents an HTML5 section with a layout based on the Bulma style library.

  3. The component has no classes, animations, or text.

  4. The content, Bulma and custom classes, and animations are passed as a JSON object.

  5. Each page detects user behavior and stores it in a Redux state.

  6. The page receives information through route parameters and query parameters.

  7. The behavior is sent to the backend along with the route and query parameters.

  8. From these parameters, the behavior history, and customer information stored in the business management system, an AI can generate a JSON that represents the content, styles, and animations according to the customer profile.

  9. CustumerJourney.js is multilingual, has light and dark themes, and animations based on the Animate.css library.

  10. CustumerJourney.js is a free, open-source library.
